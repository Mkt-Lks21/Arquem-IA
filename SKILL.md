# Arquem Data Analyst Agent

Agente de análise de banco de dados voltado para apoio à decisão de gestores de óticas e clínicas.
A skill consulta, organiza, interpreta e sintetiza dados operacionais, comerciais e financeiros, transformando queries em insights acionáveis para gestão.

## O que ele faz

* Executa consultas SQL sob demanda
* Analisa tendências de vendas, leads e atendimento
* Identifica gargalos operacionais e padrões de desempenho
* Resume resultados complexos em linguagem clara para gestores
* Gera recomendações baseadas em dados

## Para quem é

* Gerentes de loja
* Proprietários de óticas e clínicas
* Franqueados
* Diretores e coordenadores operacionais

## Benefício-chave

Reduz o tempo entre **pergunta → análise → decisão**, eliminando dependência técnica para interpretar dados.
